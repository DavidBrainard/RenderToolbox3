Dummy functions to Explain Documentation.

RenderToolbox3 functions are documented in groups of related functions called Modules. Each module gets its own page with:

  • A list of Funcitons (above). Output values appear on the left-hand side. Function names and parameters appear on the right-hand side. Function names are links to detailed documentation.
  • A Detailed Description of the module (this section).
  • Function Documentaion detailing each function (below). The documentation describes function parameters, what the function does, and what the function returns, and shows the Matlab usage syntax.
